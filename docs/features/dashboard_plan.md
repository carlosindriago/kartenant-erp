## 🎯 **MÉTRICAS Y CÁLCULOS**

### **1. Productos bajo stock mínimo:**
```php
Product::where('stock', '<=', DB::raw('min_stock'))
    ->where('status', true)
    ->select('name', 'stock', 'min_stock')
    ->orderBy(DB::raw('stock - min_stock'))
    ->limit(10)
    ->get();
```

### **2. Productos sin ventas (30 días):**
```php
Product::whereDoesntHave('saleItems', function($query) {
    $query->where('created_at', '>=', now()->subDays(30));
})
->where('stock', '>', 0)
->where('status', true)
->withCount(['saleItems as last_sale_date' => function($query) {
    $query->select(DB::raw('MAX(created_at)'));
}])
->orderBy('last_sale_date', 'asc')
->limit(10)
->get();
```

### **3. Productos más vendidos (mes):**
```php
Product::withSum(['saleItems as total_quantity' => function($query) {
    $query->whereHas('sale', function($q) {
        $q->where('created_at', '>=', now()->startOfMonth())
          ->where('status', 'completed');
    });
}], 'quantity')
->having('total_quantity', '>', 0)
->orderBy('total_quantity', 'desc')
->limit(5)
->get();
```

### **4. Ventas del día:**
```php
Sale::whereDate('created_at', today())
    ->where('status', 'completed')
    ->sum('total');
```

### **5. Tendencia de ventas (7 días):**
```php
Sale::where('created_at', '>=', now()->subDays(7))
    ->where('status', 'completed')
    ->groupBy(DB::raw('DATE(created_at)'))
    ->selectRaw('DATE(created_at) as date, SUM(total) as total')
    ->orderBy('date')
    ->get();
```

## 🎓 **LECCIONES DE DISEÑO**

### **Lo que funciona:**
✅ **Métricas con contexto**: No solo "$45,250", sino "+15% vs ayer"
✅ **Acciones claras**: "Reordenar AHORA", "Crear promoción"
✅ **Alertas priorizadas**: Rojo = urgente, Amarillo = pronto
✅ **Comparaciones**: Hoy vs ayer, vs promedio, vs meta

### **Lo que NO funciona:**
❌ Números sin contexto: "Ventas: 125"
❌ Gráficos sin conclusión: Solo líneas sin insights
❌ Métricas vanidosas: Total de productos, total de clientes (sin usar)
❌ Dashboards lentos: Más de 3 segundos de carga
